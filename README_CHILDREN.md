# نظام إدارة الأطفال - منصة التعلم

## الملفات المُنشأة:

### قاعدة البيانات:
- `database.sql` - تم تحديثه بجداول الأطفال والمهام والجلسات والشارات
- `insert_default_data.sql` - بيانات افتراضية للمواد والمهام والشارات

### الصفحات:
- `parent_dashboard.html` - لوحة تحكم الأهل
- `child_view.html` - صفحة الطفل

### ملفات CSS:
- `css/parent_dashboard.css` - تصميم لوحة تحكم الأهل
- `css/child_view.css` - تصميم صفحة الطفل

### ملفات JavaScript:
- `js/parent_dashboard.js` - وظائف لوحة تحكم الأهل
- `js/child_view.js` - وظائف صفحة الطفل

### ملفات API (PHP):
- `api/get_children.php` - جلب قائمة الأطفال
- `api/add_child.php` - إضافة طفل جديد
- `api/get_child_details.php` - جلب تفاصيل الطفل
- `api/get_current_task.php` - جلب المهمة الحالية للطفل
- `api/get_child_tasks.php` - جلب جميع مهام الطفل
- `api/get_child_badges.php` - جلب شارات الطفل
- `api/start_session.php` - بدء جلسة تعلم
- `api/complete_task.php` - إكمال مهمة ومنح شارات

## خطوات الإعداد:

### 1. إنشاء قاعدة البيانات:
في phpMyAdmin، نفّذ:
- `database.sql` - لإنشاء جميع الجداول
- `insert_default_data.sql` - لإدراج البيانات الافتراضية

### 2. تحديث dashboard.html:
تم تحديث `dashboard.html` لإضافة رابط إلى `parent_dashboard.html`

### 3. اختبار النظام:
1. سجّل الدخول من `index.html`
2. سيتم إعادة التوجيه إلى `dashboard.html`
3. اضغط على "لوحة تحكم الأهل"
4. أضف طفلاً جديداً
5. اضغط "ابدأ الجلسة" لفتح صفحة الطفل
