# تعليمات إعداد قاعدة البيانات

## خطوات الإعداد:

### 1. إنشاء قاعدة البيانات:
- افتح phpMyAdmin من خلال: `http://localhost/phpmyadmin`
- أو استخدم MySQL من سطر الأوامر
- قم بتشغيل ملف `database.sql` لإنشاء قاعدة البيانات والجدول

### 2. إعداد الاتصال:
- افتح ملف `config/database.php`
- تأكد من أن بيانات الاتصال صحيحة:
  - `$host = 'localhost'`
  - `$dbname = 'kids_learning'`
  - `$username = 'root'`
  - `$password = ''` (اتركه فارغًا إذا لم تكن هناك كلمة مرور)

### 3. اختبار النظام:
- افتح `http://localhost/kids_learning/`
- جرب إنشاء حساب جديد
- جرب تسجيل الدخول

## ملاحظات:
- تأكد من أن Apache و MySQL يعملان في XAMPP
- إذا واجهت مشاكل في الاتصال، تحقق من إعدادات MySQL في XAMPP
