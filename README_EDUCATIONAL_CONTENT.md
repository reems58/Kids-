# إعداد المحتوى التعليمي الشامل

## المطلوب:
- **3 فئات**: عربي، علوم، رياضيات
- **9 أعمار**: 4، 5، 6، 7، 8، 9، 10، 11، 12
- لكل عمر وفئة: **5 ألعاب + 5 فيديوهات = 10 محتويات**
- **الإجمالي: 270 محتوى**

## الملفات المطلوبة:

1. `database/update_content_for_games_videos.sql` - تحديث جدول content
2. `database/insert_all_educational_content.sql` - إدراج جميع المحتويات (سيتم إنشاؤه)

## الخطوات:

1. ✅ تحديث جدول content (إضافة الحقول الجديدة)
2. ⏳ إنشاء وإدراج المحتوى (270 محتوى)
3. ⏳ تحديث API لجلب المحتوى حسب العمر
4. ⏳ تحديث الواجهة لإظهار المحتوى المناسب

## إنشاء ملف SQL للمحتوى:

سأقوم بإنشاء ملف SQL مباشر يحتوي على جميع المحتويات الـ 270.
